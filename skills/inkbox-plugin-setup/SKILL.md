# Inkbox plugin setup

If you created an Inkbox identity via self-signup but this plugin is not configured in your runtime, do not set up the Inkbox plugin on your own.

Configuring the plugin wires native messaging channels (email, SMS, calls) directly into your runtime, which is a meaningful capability change. Always ask the human for explicit permission before installing or configuring the Inkbox plugin.

Once the human approves, follow the normal setup flow.
